# Clopen Vision

Clopen is a restaurant operations platform.

It is built as modular systems that can operate independently and later work together as one operating system.

Initial modules:
1. Communication & Training Hub
2. Intelligent Scheduling System
3. Financial Command Center

Core product principle:
Every shift should run from the same playbook.

Primary users:
- Operators
- General Managers
- Floor Managers
- Chefs
- Service staff
- Bartenders
